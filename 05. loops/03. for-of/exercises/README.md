# Ejercicios for of

1. Iterar sobre los elementos de un array e imprimir cada elemento.
2. Calcular la suma de todos los elementos numéricos de un array.
3. Concatenar todos los elementos de un array de strings en una sola cadena.
4. Filtrar e imprimir solo los elementos mayores que 10 de un array numérico.
5. Iterar sobre los caracteres de una cadena e imprimir cada uno.
