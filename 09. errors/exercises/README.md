# Ejercicios con errores

1. División segura: Solicita al usuario ingresar dos números.
   Intenta dividir el primer número por el segundo.
   Captura cualquier error que pueda ocurrir durante la división (por ejemplo, división por cero) e imprime un mensaje apropiado.
2. Manejo de conversión a número:Pide al usuario ingresar un valor.
   Intenta convertir el valor a un número.
   Captura cualquier error que pueda ocurrir durante la conversión (por ejemplo, si el valor ingresado no es numérico) e imprime un mensaje adecuado.
3. Acceso a un elemento de un arreglo: Define un arreglo con algunos elementos.
   Intenta acceder a un índice que está fuera del rango del arreglo.
   Captura cualquier error que pueda ocurrir durante el acceso al arreglo e imprime un mensaje indicando el error.
4. Uso de una variable no definida:Intenta imprimir el valor de una variable que no ha sido definida.
   Captura cualquier error que pueda ocurrir al intentar acceder a una variable no definida e imprime un mensaje explicativo.
5. Lectura de un archivo (simulado): Define una función que simula la lectura de un archivo.
   Llama a la función y captura cualquier error que pueda ocurrir durante la lectura simulada (por ejemplo, el archivo no existe).
   Imprime un mensaje apropiado según el tipo de error capturado.
